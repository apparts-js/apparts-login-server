Endpoints:

Add a user

v1 POST /v/1/user
  • Body:
    • email: <email>
Returns:
  • Status: 200 "ok"
  • Status: 413 { "error": "User exists" }

Login

v1 GET /v/1/user/login
  • Header:
    • Authorization: Basic btoa(uname:password)
Returns:
  • Status: 200 {
      "id": <id>,
      "loginToken": <base64>,
      "apiToken": <string>
    }
  • Status: 401 { "error": "User not found" }
  • Status: 400 { "error": "Authorization wrong" }
  • Status: 401 { "error": "Unauthorized" }

Renew API Token

v1 GET /v/1/user/apiToken
  • Header:
    • Authorization: Basic btoa(uname:token)
Returns:
  • Status: 200 <string>
  • Status: 401 { "error": "User not found" }
  • Status: 400 { "error": "Authorization wrong" }
  • Status: 401 { "error": "Unauthorized" }

Get a user

v1 GET /v/1/user
  • Header:
    • Authorization: Basic btoa(uname:token)
Returns:
  • Status: 200 {
      </>: </>
    }
  • Status: 401 { "error": "User not found" }
  • Status: 400 { "error": "Authorization wrong" }
  • Status: 401 { "error": "Unauthorized" }

Delete a user

v1 DELETE /v/1/user
  • Header:
    • Authorization: Basic btoa(uname:password)
Returns:
  • Status: 200 "ok"
  • Status: 401 { "error": "User not found" }
  • Status: 400 { "error": "Authorization wrong" }
  • Status: 401 { "error": "Unauthorized" }

Update a user

Currently, only updating the password is supported.

v1 PUT /v/1/user
  • Header:
    • Authorization: Basic btoa(uname:token)
  • Body:
    • ? password: <password>
Returns:
  • Status: 400 { "error": "Nothing to update" }
  • Status: 400 { "error": "Password required" }
  • Status: 200 {
      "id": <id>,
      "loginToken": <base64>,
      "apiToken": <string>
    }
  • Status: 401 { "error": "User not found" }
  • Status: 400 { "error": "Authorization wrong" }
  • Status: 401 { "error": "Unauthorized" }

Reset the password

v1 POST /v/1/user/:email/reset
  • Params:
    • email: <email>
Returns:
  • Status: 404 { "error": "User not found" }
  • Status: 200 "ok"
